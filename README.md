### HTEffect 版本说明
<b>Version: 1.2.0</b><br>
* 优化精细磨皮算法，提升清晰度；
* 优化功能模块分类；
* 开通人脸检测和关键点识别API；
* 新增美肤、美型参数的极大值限定接口；

<b>Version: 1.1.0</b><br>
* 增加人脸关键点追踪功能；
